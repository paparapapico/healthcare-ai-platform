# 🏥 AI Healthcare Platform

## 프로젝트 비전
최고 수준의 AI 기반 통합 헬스케어 플랫폼 구축

## 핵심 기능
- 🏃 AI 운동 자세 분석
- 😴 수면/스트레스 관리  
- 👵 시니어 건강 관리
- 🩺 만성질환 관리
- 👩 여성 건강 관리

## 기술 스택
- Backend: FastAPI, PostgreSQL
- AI/ML: TensorFlow, MediaPipe
- Mobile: React Native
- Infra: AWS, Docker
